# Webchat

## Что реализовано:

- Регистрация (ui/auth)
- Локализация
- Создание:
    - Миграции
    - Модели
    - Связь "Один ко многим"
    - Формы в Bootstrap
    - POST запрос

## Планируется реализовать:

- Станица пользователя
- Аватарка
- Real-time (WebSockets скорее всего)
- Ответы на сообщения
- Редактирование сообщения
